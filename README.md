### Simple LOGGER with Express JS and Mongoose ###

![Screenshot](screenshot.png)

*Default port is 8888*

*Default mongo port is 37017*

Install
=======
npm start

For developer, start server
=======
npm run start:dev

For production, start server
=======
npm start

API usage
=======
POST /logger
{
  "msg": "message for logging",
  "type": "info"
}

GET /logger
